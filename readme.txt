Show random images from the media manager on your website. As many as you want. No gallery plugin required.

== Description ==

RandImage shows a number of images randomly in the sidebar. Offers simple customization and manually set amount of images. Now also comes with a widget built-in.

**Features**

* Show random images
* Choose how many images you want to show
* Link to the gallery image or post the image is in
* Simple in use
* Integrates with WordPress's Media Manager
* Widget included

== Installation ==

1. Unzip the file and upload it to your /wp-content/plugins/ folder.
1. Activate from the plugins page.
1. Insert this `<?php randimage_sidebar(); ?>` in your theme where you want to show the images.
1. Or activate the widget.
